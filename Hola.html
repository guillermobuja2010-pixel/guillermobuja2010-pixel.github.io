Inicio

En esta segunda fase del proyecto se llevó a cabo la comparación de la temperatura entre una vivienda que tiene aislante térmico y una vivienda convencional para observar, a lo largo del tiempo, el comportamiento térmico de ambas viviendas. Esta comparación se realizó a partir de una tabla de datos con valores numéricos y una gráfica con sus correspondientes coordenadas en el plano cartesiano, permitiendo observar la variación de la temperatura interior de las viviendas a partir del uso de unos determinados materiales aislantes para mantener la temperatura interior.

Clima

El experimento se realizó considerando un clima desértico, caracterizado por temperaturas altas durante el día, poca humedad y una fuerte exposición al sol. Este tipo de clima provoca que las viviendas acumulen mucho calor, por lo que el uso de materiales aislantes es importante para mantener una temperatura interior más estable y cómoda.

Materiales

  • Barras refelctivas
  • Laminas aluminizada
  • Propiedades
  • Barras reflectivas
  • Lija
  • Cuter
  • Regla
  • Pasto sintetico

Metodología

Para realizar el experimento, se construyeron dos viviendas pequeñas utilizando los mismos materiales y medidas. Una de las viviendas fue elaborada con materiales aislantes, como barras reflectivas y pasto sintético, mientras que la otra se dejó como vivienda convencional. Después, ambas viviendas fueron colocadas bajo las mismas condiciones de exposición al sol en un clima desértico.

Posteriormente, se midió la temperatura interior de cada vivienda durante diferentes periodos de tiempo y los datos obtenidos fueron registrados en una tabla. Finalmente, los resultados se representaron en una gráfica con coordenadas en el plano cartesiano para comparar el comportamiento térmico de ambas viviendas y analizar la efectividad del aislante térmico.

Conclusión

Al comparar las viviendas, se observa que la casa con aislante térmico presenta un mejor funcionamiento a la hora de controlar la temperatura. Esto acredita que los materiales aislantes funcionan correctamente para climas extremos, en especial para las zonas cálidas que contemplan climas desérticos.
